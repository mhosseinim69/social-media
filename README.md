## SocialMedia

SocialMedia is a versatile and customizable social networking platform designed to connect people and facilitate meaningful interactions online. Built with modern technologies, it offers a robust set of features for users to share updates, connect with friends, and engage with communities.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
# run and build
$ docker-compose up -d

```

## Swagger

```bash
# run on browser
$ http://localhost:3000/api

```
